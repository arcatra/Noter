~~~~~~~~~~~~~~ help ~~~~~~~~~~~~~~
commands:

-new : Add new task to the task pool.
-remove : Remove a task from the task pool.
-update : Update a task with NEW NAME or DESCRIPTION.
-done : Mark a task as DONE, this will remove the task from tasks pool.
-doneall : Mark all of the tasks as DONE, this will remove all the tasks from tasks pool.

-help : to get help about a spesific command or help.
-about : More info about Noter.

Usage:

-new "<Name>:<Description>"
-remove <task ID: int>
-update <task ID: int> "<new name> "<new description>"
-done <task ID: int>
-doneall

-help
-about
