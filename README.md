# Noter
A daily tasks management tool, for productivity and clarity.
Keeps your tasks local.

## Commands
- new : To create a new task.
- display : To display the tasks in task pool, null if none is present.
- remove : To remove/delete the task, with task ID.
- update : To update an exixting task/tasks.
- help : For help.
- about : To learn more about Noter.

### Options
- --task<task name> : Provide the new task name, this should be followed by 'new' command if creating a new task.
- --desc<description> : Provide the description for the task given, will defalut to 'No desc'\
                      if nothing is provided, and this should also be followed by --task \
                      and 'new' command if creating a new task.
